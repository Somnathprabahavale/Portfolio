# Portfolio Website

A simple, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- Contact form with basic validation
- Skills showcase
- Projects section
- Clean, modern design

## How to View

1. Open `index.html` in your web browser
2. Navigate through the sections using the menu or scroll

## Customization

- Replace "John Doe" with your actual name in `index.html`
- Update the about section with your personal information
- Modify the skills list to match your expertise
- Add your project details in the projects section
- Update contact information
- Replace `profile.jpg` with your actual profile picture (200x200px recommended)

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - CSS styling
- `script.js` - JavaScript for interactivity

## Browser Support

Works in all modern browsers including Chrome, Firefox, Safari, and Edge.

## Troubleshooting

- If images don't load, ensure the image files are in the same directory as `index.html`
- For local development, you may need to run a local server if using certain features
- If the form doesn't submit, check that JavaScript is enabled in your browser
